📚 API Endpoints

GET /
Get basic API information and status. No authentication required.
POST /stack-master/chat
Chat with The Stack Master AI Coach. Get personalized financial advice with authentic cultural voice. Free tier uses Qwen model.
POST /stack-master/analyze-stack
Get comprehensive financial analysis with personalized recommendations. Uses free Qwen model for culturally-aware guidance.
POST /premium/finrobot-agents
Access advanced FinRobot agents for professional-grade financial analysis. Requires $9.99 beta subscription.
GET /subscription/plans
Get available subscription plans and pricing information. Compare free and premium tiers.
POST /investment/advice
Get personalized investment advice using the free Qwen AI model. Culturally-aware guidance with actionable steps.
POST /credit/building-plan
Create a personalized credit building plan using the free Qwen AI model. Step-by-step guidance to improve credit scores.
GET /community/stack-challenges
Get available community challenges for financial growth. Includes both free and premium challenges.
GET /education/topics
Get available financial education topics and resources. Includes both free and premium educational content.
